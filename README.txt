Заходим на https://discord.com/developers/applications
Нажимаем New Application и даем название нашему чуду
(Можете поменять иконку, если хотите)
Заходим во вкладку Bot, нажимаем Add bot и Yes, do it!
В той же вкладке нажимаем COPY, чтобы скопировать токен бота
Запишите этот токен, например, в текстовый файл.

Далее нам нужно пригласить бота
Заходим во вкладку OAuth2
Нажимаем bot и копируем ссылку
Переходим по ссылке и приглашаем бота на сервер
Все! Бот на сервере.

Теперь переходим ко 2 части "Канал и цель"
Заходим в настройки пользователя
Внешний вид
Снизу будет режим разработчика. Включите его.
Теперь по пкм мы будем иметь возможность копировать ID всего, что есть в Discord.

Выбираем чат, нажимаем пкм и копировать ID
Этот ID записываем там же, где и токен бота
Теперь делаем то же самое с целью (пользователем).

Когда все будет выполнено, нужно открыть программу и отправить токен, id канала и id цели
После этого бот запустится и будет постоянно (пока работает программа) пинговать по 5 сообщений пользователя
